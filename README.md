~# Linux User & Access Management System

# Project Summary
This project demonstrates role-based access control (RBAC) on a Linux server hosted on AWS EC2. 
Multiple users and groups were created to control access to directories and administrative privileges.
The project focuses on Linux security, user management, and enterprise-level access control.

# Problem Statement
Organizations must manage multiple users securely while preventing unauthorized access.

# Solution
Implemented Linux users and groups with controlled permissions, restricted directory access, 
and sudo privileges for administrative users.

- Linux user & group management
- File and directory permissions
- Security administration
- AWS EC2
